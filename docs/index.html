About#

本研究室では、微視的な理論に立脚し、数値解析および計測技術を融合した総合的なアプローチにより、異なる長さスケールを有するマルチスケール構造体の熱や電気などの輸送特性や物性の理解と制御を目指します。さらに、ナノマイクロ領域で顕在化する現象を活用し、機械工学の立場から次世代のエネルギー・エレクトロニクス産業の発展に貢献する新たな学理の開拓と、革新的な材料やデバイスの創成に取り組んでいます。
